Chat App

This React Native mobile app uses Firebase/Firestore for real-time data storage and offline capabilities, and React Native Gifted Chat for an intuitive chat interface. The app also uses React Navigation for screen transitions, React Native Maps for geolocation, AsyncStorage for local data storage, and Expo for development and testing in Android and iOS.

This app was built in July, 2024.

Meet App

The Meet App is an event management tool built using React and deployed as a PWA. It integrates Google Calendar API for real-time event data retrieval, AWS Lambda for serverless processing, and secure user authentication via Google OAuth 2.0. Recharts provides users with dynamic data visualization. Testing is facilitated by Puppeteer and Jest.

This app was built in June, 2024.

MyMovies App

This React app features a dynamic search functionality powered by Fuse, and supports CRUD operations for managing user data through a custom API. React Bootstrap and SCSS ensure a beautiful and user-friendly interface, while Parcel handles bundling. The backend is built in MyMovies-Server Repo using Node.js, Express, and MongoDB Atlas.

This app was built in April, 2024.

Lists App

This React Native and Expo-based mobile app manages shopping lists. Users can create, view, and manage their shopping lists with offline functionality (AsyncStorage) and automatic data syncing when online (Firebase Firestore).

This app was built in July, 2024.

Planting by Zones App

The Planting by Zones app allows users to discover their local hardiness zone and access a tailored planting calendar by simply entering a zipcode. Using HTML for structure and JavaScript for frontend functionality, the app integrates a third-party API to fetch and display zone-specific information. The responsive UI is built with CSS and Bootstrap.

This app was built in March, 2024.

To Do List App

This ReactJS app, rebuilt from a vanilla JavaScript todo-list-app, integrates Redux for robust state management. Key functionalities include adding, toggling, and deleting todo items, with support for keyboard input to swiftly add tasks. This app employs Bootstrap-React for a responsive user interface, and uses Parcel as a bundler.

This app was built in May, 2024.
