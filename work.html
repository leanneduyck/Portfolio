Meet App

The Meet App is an event management tool built using React and deployed as a PWA, ensuring seamless offline functionality. It uses Google Calendar API integration for real-time event data retrieval, AWS Lambda for serverless processing, and secure user authentication via Google OAuth 2.0. Recharts improves user experience with dynamic data visualization. Puppeteer and Jest facilitate testing, ensuring reliability across different testing scenarios.

This app was built in June, 2024.

MyMovies App

The MyMovies App is built using React and deployed on Netlify. It features a user-friendly interface with dynamic search functionality powered by Fuse, allowing users to browse a movie collection, and it supports CRUD operations for managing user data through the MyMovies-Server API. React Bootstrap and SCSS enhance the visual appeal and responsiveness, while Redux manages state and Parcel handles bundling; the backend is built with Node.js, Express, and MongoDB Atlas, and is deployed on Heroku.

This app was built in April, 2024.

To Do List App

This ReactJS application, rebuilt from a vanilla JavaScript todo-list-app, integrates Redux for robust state management. Key functionalities include adding, toggling, and deleting todo items, with support for keyboard input to swiftly add tasks. This app employs Bootstrap-React for a responsive user interface, complemented by deployment on Netlify for seamless continuous integration and hosting, demonstrating proficiency in ReactJS, Redux, Parcel bundler, React Bootstrap, and deployment workflows.

This app was built in May, 2024.
